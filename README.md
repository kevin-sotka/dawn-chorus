# Dawn Chorus

It is 5am in a watercolor forest. Echo each bird's song, branch by branch, and wake the forest into a sunrise symphony you built yourself.

**Play it:** https://kevin-sotka.github.io/dawn-chorus/

## How to play
Listen: a bird sings and its branches light in order. Your turn: tap the branches in the same order. Every correct phrase adds that bird to the looping chorus and lifts the sun one shade higher. Five birds, one dawn, no way to lose, only prettier and uglier paths to sunrise.

## Tech
One self-contained `index.html`. Canvas 2D watercolor sky, five distinct synthesized birdsong voices in WebAudio with a shared lookahead scheduler. Real bird recordings from xeno-canto (CC licensed, credited) and licensed Adobe Stock photos now feature in the field-guide cards and the chorus. The game is still dependency-free code plus these local assets, nothing to install, nothing fetched from a server.

Part of [The Aviary](https://kevin-sotka.github.io/the-aviary/), three bird games competing for your vote. Built by the SodClaw Game Foundry.
